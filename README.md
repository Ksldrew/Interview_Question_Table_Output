# Table Output & Processing

This project displays a data table (Table 1) and calculates a second table (Table 2) based on predefined rules.


## Description
- Table 1 contains input values
- Table 2 displays calculated results based on the rules:
  - Alpha = A5 + A20
  - Beta = A15 / A7
  - Charlie = A13 * A12

All calculations are performed using JavaScript.
